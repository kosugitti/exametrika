# internal functions for PSD of Item parameters

internal functions for PSD of Item parameters

## Usage

``` r
PSD_item_params(model, Lambda, quadrature, marginal_posttheta)
```

## Arguments

- model:

  2,3,or 4PL

- Lambda:

  item parameters Matrix

- quadrature:

  quads

- marginal_posttheta:

  marginal post theta
